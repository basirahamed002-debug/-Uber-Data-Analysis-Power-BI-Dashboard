# 🚗 Uber Data Analysis — Power BI Dashboard

An interactive Power BI dashboard analyzing Uber's 2025 ride data across 6 vehicle types.

## 📊 Dashboard Overview

![Dashboard Preview](dashboard_preview.png)

## 🔑 Key Metrics
- **93K** Completed Bookings
- **57K** Lost Bookings  
- **₹52M** Total Revenue
- **24.64** Avg Trip Distance
- **4.23** Avg Driver Rating

## 📌 Insights Covered
- Revenue breakdown by vehicle type (Auto leads at ₹13M)
- Monthly vs Quarterly completed booking trends
- Customer & driver rating analysis
- Booking value distribution across time periods
- Vehicle-wise performance comparison (Auto, Bike, Go Mini, Go Sedan, Premier Sedan, Uber XL)

## 🛠️ Tools Used
- **Power BI Desktop** — Dashboard & DAX measures
- **Microsoft Excel** — Source data
- **DAX** — Custom KPI calculations

## 💡 Key Findings
- Auto rickshaws generate the highest revenue despite lower per-trip fare
- Significant lost bookings (57K) indicate supply-demand gap opportunities
- Driver ratings average 4.23, suggesting consistent service quality
